SDIF Utilities

The SDIF Utilities folder contains a set of tools to convert and inspect SDIF files. Most of these utilities are also available as sourcecode and for different platforms. For all information about the SDIF Open Standard, please visit http://sdif.sourceforge.net 

SDIFConverter README

SDIFConverter can convert certain types of SDIF files to text files. This can be useful to import sound analysis data into applications that don't support the SDIF standard (like CSound, Excel, etc.).
